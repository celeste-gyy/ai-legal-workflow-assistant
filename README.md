# AI Legal Workflow Assistant

An enterprise legal workflow assistant for initial contract and compliance risk review.

The app helps business teams capture a business request, run an initial review, route the matter by risk level, and surface legal review recommendations for high-risk scenarios.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Environment

DeepSeek credentials should stay in local environment files such as `.env.local`.

Required variables:

```bash
DEEPSEEK_BASE_URL=
DEEPSEEK_API_KEY=
DEEPSEEK_MODEL=
```
